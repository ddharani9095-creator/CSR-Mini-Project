CSR Impact Analysis Dashboard is a Power BI project designed to analyze Corporate Social Responsibility initiatives and evaluate their impact across different sectors, regions, and beneficiary groups.

The project focuses on transforming raw CSR data into meaningful business insights through data cleaning, data modeling, DAX calculations, and interactive visualizations. The dashboard helps stakeholders monitor CSR spending, measure project effectiveness, track beneficiary reach, identify funding trends, and support data-driven decision-making.

Key Analysis Areas:
• CSR Budget Allocation and Utilization
• Sector-wise CSR Investment Analysis
• Regional Performance Tracking
• Project Completion and Outcome Monitoring
• Year-over-Year CSR Trend Analysis

Tools Used: Excel | Power BI | Power Query | DAX | Data Modeling
